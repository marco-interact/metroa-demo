# Cursor Logs

This directory contains all Cursor AI-generated documentation, session logs, and development notes organized by date.

## Structure

```
cursor-logs/
├── README.md (this file)
├── archive/   (historical logs from previous development sessions)
└── YYYY-MM-DD/   (current session logs organized by date)
```

## Purpose

- Track development progress and decisions
- Document deployment configurations
- Archive setup scripts and commands
- Maintain session summaries and status reports

## CI/CD Integration

This folder is automatically organized by date and included in the repository for:
- Development history tracking
- Knowledge transfer between team members
- Debugging and troubleshooting reference
- Deployment configuration archive
